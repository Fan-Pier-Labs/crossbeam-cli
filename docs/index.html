Why this exists

Crossbeam's API is locked behind its top tier.

Crossbeam is the account-mapping platform that shows where your customers and prospects overlap with your partners'. But getting that data out programmatically — to pipe into a spreadsheet, a script, or an AI assistant — officially requires the Supernode plan. Everyone else clicks through the web UI, 50 records at a time.

This project uses the same endpoints the Crossbeam web app already calls when you're signed in. So anyone with a login — including the free plan — can pull their own data with code.

How it works

Your credentials never leave your machine.

No backend, no proxy, no telemetry. It talks directly to Crossbeam over HTTPS — that's it.

1

You log in

Provide your own Crossbeam email and password. The tool authenticates against auth.crossbeam.com exactly like the web app.

2

Session stays local

The session cookie is cached at ~/.crossbeam on your own disk. Nothing is sent anywhere else — no "phone home."

3

You read your data

Every call goes straight to api.crossbeam.com. Ask Claude in plain English, or pipe JSON in the terminal.

For developers

Command-line tool

A Node package and CLI. Every command emits JSON, with a raw escape hatch for any endpoint. Perfect for scripts, cron jobs, and coding agents.

# install npm install -g crossbeam-cli # your Crossbeam login export CROSSBEAM_USER=you@example.com export CROSSBEAM_PASS=... # read your partners as JSON crossbeam partners list --json
  • Importable Node/TypeScript client too
  • Built for Claude Code, Cursor & other agents
